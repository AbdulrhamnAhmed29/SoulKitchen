# 🍽️ SOUL KITCHEN — Modern Restaurant & E-Commerce Platform

---

## 🌐 Overview

**SOUL KITCHEN** is a high-end, full-stack restaurant ecosystem that bridges premium e-commerce functionality with an advanced reservation management system.

The platform delivers a seamless experience for users to explore menus, manage digital carts, and book tables — supported by a personalized dashboard for tracking orders and reservations.

Built with a strong focus on **performance, scalability, and modern frontend architecture**.

---

## ✨ Core Features

### 🧑‍🍳 User Experience

* Dark-mode luxury UI with high-performance landing & about pages
* Dynamic menu catalog with real-time updates
* Server-side sorting and optimized data rendering
* Google Authentication for seamless onboarding

---

### 🛒 Advanced Ordering System

* Persistent cart (Add / Remove / Update items)
* Smooth checkout flow
* Receipt-style checkout UI
* Printable PDF receipt generation

---

### 🍴 Table Reservation System

* Integrated booking system
* Reservation history tracking
* Clean and structured booking flow

---

### 📊 Personalized Dashboard

* Centralized user hub
* Order history tracking with statuses
* Reservation management overview

---

## 🧱 Tech Stack

| Layer            | Technology                                |
| ---------------- | ----------------------------------------- |
| Frontend         | React.js, Tailwind CSS                    |
| State Management | TanStack Query (React Query)              |
| Form Handling    | React Hook Form                           |
| Backend          | Strapi (Headless CMS)                     |
| UI Utilities     | SweetAlert2, React Icons, React Hot Toast |

---

## 🏗️ Engineering & Architecture

* Feature-Based Architecture (FBA) for full scalability
* Domain-based structure for clean separation of concerns
* Centralized API layer for maintainability
* Server-state management using React Query
* Optimized caching and background refetch strategy
* Reusable and modular component system

---

## 🚀 Performance & Scalability

* Server-side pagination and sorting for large datasets
* Reduced API overhead via intelligent caching
* Background data synchronization
* Optimized rendering and UI updates

---

## 🔄 Development Workflow

* Git Flow strategy (feature branches per task)
* Pull Request-based development
* Clean and descriptive commit history
* Continuous self code review before merging

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/AbdulrhamnAhmed29/SoulKitchen?tab=readme-ov-file
```

---

### Install Dependencies

```bash
npm install
```

---

### Environment Setup

Create a `.env` file and add your Strapi URL:

```env
REACT_APP_URL=your_strapi_url
```

---

### Run Project

```bash
npm run dev
```
